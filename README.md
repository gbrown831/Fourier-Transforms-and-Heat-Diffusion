# Fourier-Transforms-and-Heat-Diffusion
Code implementation of Green's function and Fast Fourier Transform to solve the 1D heat-diffusion equation. These implementations were applied to synthesized data and real temperature data from the Curiosity Rover on Mars.

Link to dataset: https://www.kaggle.com/datasets/peijenlin/msl-m-rems-2-edr-v1.0

This project was done for a 400-level Math class as College of William and Mary. Mathematical proofs and derivations are done in the paper. All the code done for this project was done in one Jupyter Notebook file. Results can be reproduced as long as the TAB files are downloaded and their respective hard-coded file paths are edited.
